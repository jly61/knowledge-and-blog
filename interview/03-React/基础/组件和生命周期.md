# React 组件和生命周期

## 函数组件

### 基本语法
```jsx
function MyComponent(props) {
  return <div>{props.name}</div>
}
```

### Hooks
- useState
- useEffect
- useContext
- useMemo
- useCallback

## 类组件

### 基本语法
```jsx
class MyComponent extends React.Component {
  render() {
    return <div>{this.props.name}</div>
  }
}
```

### 更新阶段
- render

### 卸载阶段
