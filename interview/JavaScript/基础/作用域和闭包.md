# 作用域和闭包

## 作用域

### 全局作用域
在函数外部定义的变量，可以在任何地方访问。

### 函数作用域
在函数内部定义的变量，只能在函数内部访问。

### 块级作用域
ES6 引入的 `let` 和 `const` 创建的块级作用域。

## 闭包

### 定义
函数能够访问其外部作用域的变量，即使外部函数已经执行完毕。

### 示例
```javascript
function outer() {
  let count = 0
  return function inner() {
    count++
    return count
  }
}

const counter = outer()
console.log(counter()) // 1
console.log(counter()) // 2
```

### 应用场景
- 数据私有化
- 函数工厂
- 模块化
