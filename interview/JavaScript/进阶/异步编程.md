# 异步编程

## Promise

### 基本用法
```javascript
const promise = new Promise((resolve, reject) => {
  if (success) {
    resolve(value)
  } else {
    reject(error)
  }
})
```

### Promise 链式调用
```javascript
promise
  .then(value => {
    // 处理成功
  })
  .catch(error => {
    // 处理错误
  })
```

## async/await

### 基本用法
```javascript
async function fetchData() {
  try {
    const data = await fetch('/api/data')
    return data.json()
  } catch (error) {
    console.error(error)
  }
}
```
